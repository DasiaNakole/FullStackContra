# Portfolio Case Study

## Title

Built a Full-Stack Business Dashboard

## Summary

I built a full-stack dashboard for managing lead data, visualizing metrics, and exposing business records through API endpoints.

## Problem

Businesses often need lightweight internal tools to track data and view performance without paying for or configuring a full enterprise platform.

## Solution

I created a dashboard with:

- login authentication
- analytics cards
- searchable lead table
- API routes for metrics and records
- SQLite-backed seeded data

## Outcome

The result is a portfolio-ready internal tool that demonstrates end-to-end full-stack development, from database design through backend routes to frontend UI.
