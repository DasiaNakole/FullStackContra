# Contra Listing Copy

## Title

Full-Stack Business Dashboard

## Short Pitch

I build internal tools and SaaS-style dashboards that help businesses log in, manage data, and visualize performance in one place.

## Description

This project is a full-stack business dashboard built for tracking lead activity and core pipeline metrics.

It includes a login flow, analytics cards, a searchable table, backend API endpoints, and a SQLite-backed data layer. The goal was to create a practical internal tool that feels like a lightweight SaaS product instead of a static frontend demo.

## Deliverables

- login page
- dashboard UI
- Express backend
- API endpoints
- database-backed records
- analytics and reporting components
